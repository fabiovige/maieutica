❓ Perguntas Frequentes (FAQ)

Respostas para as dúvidas mais comuns sobre o sistema Maiêutica.

🔐 Login e Acesso

Esqueci minha senha, como recupero?

Na tela de login, clique em "Esqueci minha senha". Digite seu email cadastrado e você receberá um link para criar nova senha.

Posso acessar o sistema de qualquer dispositivo?

Sim! O sistema é responsivo e funciona em desktop, tablet e celular. Recomendamos desktop para avaliações e tablet/celular para visualização de resultados.

Como altero minha senha?

Clique no seu avatar (canto superior direito) → "Alterar Senha" → Digite senha atual e nova senha → Salvar.

👶 Crianças

Posso deletar uma criança permanentemente?

Não. O sistema utiliza soft delete (exclusão lógica). A criança sai da lista ativa mas seus dados permanecem no banco. Administradores podem recuperar se necessário.

Uma criança pode ter mais de um profissional responsável?

Sim! Ao cadastrar ou editar uma criança, você pode atribuir múltiplos profissionais (útil para equipes multidisciplinares).

📋 Avaliações

Preciso avaliar todas as competências em uma sessão?

Não! Avalie apenas as competências relevantes para o caso. O ideal é avaliar 10-15 competências por sessão (cerca de 1 hora).

Posso editar uma avaliação após finalizá-la?

Sim, mas apenas até 24 horas após a finalização. Depois disso, apenas administradores podem editar (para manter integridade dos dados históricos).

O que significa nota 0?

Nota 0 = "Não testado" (a competência não foi avaliada naquela sessão). NÃO significa que a criança não sabe!

Perdi uma avaliação que estava preenchendo. O que faço?

Se salvou como rascunho, acesse "Avaliações" → "Rascunhos" e continue de onde parou. Se não salvou, infelizmente os dados foram perdidos (o sistema não salva automaticamente).

📊 Resultados e Gráficos

Como interpreto um percentual de 55%?

55% está na faixa amarela (41-70%), indicando desenvolvimento em progresso. Continue estimulando, mas não é prioridade máxima de intervenção.

Por que o gráfico não aparece?

Certifique-se que: (1) A avaliação foi finalizada (não está em rascunho), (2) Pelo menos 3 competências foram avaliadas, (3) Seu navegador tem JavaScript habilitado.

Posso comparar 3 ou mais avaliações?

Atualmente o sistema compara automaticamente primeira vs. última. Para comparações mais complexas, exporte os PDFs e analise manualmente.

📝 Planos

Quantas competências devo incluir em um plano?

Recomendamos 3-5 competências. Planos muito extensos (10+) raramente são cumpridos integralmente.

Como o sistema sugere competências para o plano?

O sistema analisa a última avaliação e sugere automaticamente competências com nota ≤ 1 (emergente ou não testado).

📄 Relatórios PDF

O PDF não está sendo gerado. O que faço?

Verifique: (1) Avaliação está finalizada, (2) Navegador permite downloads, (3) Não há bloqueador de pop-ups ativo. Se persistir, limpe o cache do navegador.

Posso personalizar o layout do PDF?

Não. O layout é padronizado para garantir profissionalismo. Se necessário, edite o PDF externamente (Adobe Acrobat, etc.).

🔧 Problemas Técnicos

O sistema está lento. O que faço?

Tente: (1) Limpar cache do navegador (Ctrl+Shift+Del), (2) Atualizar página (F5), (3) Usar Chrome ou Firefox (melhores suportados).

Meus dados estão seguros?

Sim! O sistema segue LGPD, usa criptografia, backup diário e controle de acesso por permissões. Dados de saúde são tratados com máxima segurança.

O sistema funciona offline?

Não. É necessário conexão com internet para acessar o sistema.
